# Тестовое задание на позицию Junior System Analyst

## Описание

В репозитории представлены решения тестового задания на позицию Junior System Analyst.

В рамках выполнения задания были решены следующие задачи:

1. Анализ и уточнение требований к функционалу корзины интернет-магазина.
2. Проектирование REST API для экрана магазинов-партнеров мобильного приложения.
3. Проектирование верхнеуровневой архитектуры системы PUSH-уведомлений.

---

## Структура репозитория

### task-1-requirements-analysis

Содержит:

* анализ требований;
* выявленные противоречия и неоднозначности;
* вопросы бизнес-заказчику;
* ER-диаграмму сущностей корзины.

### task-2-api-design

Содержит:

* описание REST API;
* пример HTTP-запроса;
* пример JSON-ответа;
* описание структуры данных;
* обработку возможных ошибок.

### task-3-architecture

Содержит:

* описание архитектуры PUSH-уведомлений;
* описание компонентов системы;
* сценарий доставки уведомлений;
* архитектурную диаграмму.

---

## Использованные артефакты

* Use Case Analysis
* Requirements Analysis
* REST API Design
* ER Diagram
* High-Level Architecture Diagram
